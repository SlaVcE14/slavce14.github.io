Privacy Policy for Json List

Last updated: 24.10.2025

Information Collection and Use

Json List does not automatically collect, store, or share any personal data from users. The app may access basic device information (such as screen size, model, and manufacturer) only for display and debugging purposes. This information is stored locally on your device and is never transmitted automatically. Users may choose to manually share logs or data via email for support or feedback. This action is completely voluntary.

Third-Party Libraries

Json List uses the following libraries:

  • OkHttp – used only for optional, user-initiated network requests.
  • Gson – used locally for JSON parsing.
These libraries do not collect or share any personal data on their own.

Json List also uses various open-source UI libraries to provide dialog and interface components. These libraries operate entirely on-device and do not collect, store, or transmit any user data.

Data Sharing

The app does not share user data with any third parties.

Children’s Privacy

Json List is intended for users aged 16 and over and does not target children under 16.

Changes to This Policy

If there are any updates to this privacy policy, they will be reflected on this page.
